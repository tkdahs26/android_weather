# Weather Forecast App (Android)

## 프로젝트 개요

공공 데이터 활용 날씨 API를 활용해 날씨 예측 정보를 제공하는 Android 기반 날씨 애플리케이션입니다.

선택한 지역의 현재 날씨와 기상 예보를 조회해 사용자에게 화면으로 표시하도록 구현했습니다.

---

## 주요 기능

### 날씨 정보 조회
선택한 지역의 현재 날씨 정보 표시
기온, 날씨 상태 등 기본적인 기상 정보 제공

### 날씨 예측 정보
 시간대 또는 날짜별 날씨 예측 정보 조회
 API 응답 데이터를 화면에 출력



## 구현 특징

Android 앱에서 날씨 API 호출
JSON 형식의 응답 데이터를 반영
조회 지역 변경 가능



## 사용 기술

- Android (Java)
-공공데이터 API
- JSON
